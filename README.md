# website6810110055

Basic Flask web app.

Run:

1. Create and activate a virtualenv:

```powershell
python -m venv venv
.\venv\Scripts\Activate
```

2. Install requirements:

```powershell
pip install -r requirements.txt
```

3. Run the app:

```powershell
python app.py
```

Then open http://127.0.0.1:5000
